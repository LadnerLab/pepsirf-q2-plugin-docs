q2-autopepsirf
==============

Qiime2 plugin used for the automation of q2-pepsirf and
q2-ps-plot.

Installation
------------

Dependecies:

- ``qiime2-2021.11``
- ``q2-pepsirf1``
- ``q2-ps-plot``
- ``PepSIRF``

All installation information can be located here:
https://github.com/LadnerLab/q2-autopepsirf.git

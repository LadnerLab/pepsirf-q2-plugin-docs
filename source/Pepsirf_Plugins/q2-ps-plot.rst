q2-ps-plot
==========

Qiime2 Plug-in for the creation of visualizations 
from PepSIRF outputs.

Installation
------------

Dependecies:

- ``qiime2-2021.11 +``
- ``q2-pepsirf1``
- ``PepSIRF``
- ``altair``
- ``altair_saver``
- ``jsonschema 3.2``

All installation information can be located here:
https://github.com/LadnerLab/q2-ps-plot

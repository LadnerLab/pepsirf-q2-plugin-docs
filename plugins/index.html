QIIME 2 microbiome analysis functionality is made available to users via plugins. The following official plugins are currently included in QIIME 2 train releases:

  • autopepsirf
    • Pipelines
      • diffEnrich: diffEnrich Pepsirf Pipeline
      • diffEnrich-tsv: diffEnrich tsv Pepsirf Pipeline
  • pepsirf
    • Methods
      • bin: pepsirf bin module
      • deconv-batch: pepsirf deconv batch mode module
      • deconv-singluar: pepsirf deconv singular mode module
      • demux: pepsirf demux module
      • enrich: pepsirf enrich module
      • infoSNPN: pepsirf info (samples/peptide names) module
      • infoSumOfProbes: pepsirf info (sum of probes) module
      • link: pepsirf link module
      • norm: pepsirf norm module
      • subjoin: pepsirf subjoin module
      • zscore: pepsirf zscore module
  • ps-plot
    • Pipelines
      • zenrich-tsv: zenrich TSV Pipeline
    • Visualizers
      • enrichmentRCBoxplot: Enriched Read Counts BoxPlot
      • readCountsBoxplot: Read Counts BoxPlot
      • repScatters: Rep Scatter
      • zenrich: Z Enrichment Variance Visualizer

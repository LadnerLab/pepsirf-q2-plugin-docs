ps-plot¶

Description Qiime2 Plug-in for the creation of visualizations from PepSIRF outputs.
Version 2024.4.29+57.g8bc9f3a
Website https://github.com/LadnerLab/q2-ps-plot
Support Please post to the QIIME 2 forum for help with this plugin: https://forum.qiime2.org
q2cli Invocation 
qiime ps-plot
Artifact API Import 
from qiime2.plugins import ps_plot

Pipelines¶

  • mutantScatters-tsv: Mutant Scatters
  • proteinHeatmap-dir: Protein Alignment Heatmap Dir
  • repScatters-tsv: Rep Scatter TSV
  • zenrich-tsv: zenrich TSV Pipeline

Visualizers¶

  • enrichmentRCBoxplot: Enriched Read Counts BoxPlot

  • proteinHeatmap: Protein Alignment Heatmap
  • readCountsBoxplot: Read Counts BoxPlot
  • repScatters: Rep Scatter
  • volcano: Volcano Visualizer
  • zenrich: Z Enrichment Variance Visualizer
  • zscatter: Z Score Scatter Visualization
